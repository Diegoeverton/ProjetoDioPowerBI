# Projeto Dio PowerBI
Projeto referente a criaçao de uma dashboard com Power Bi
Neste Dashboard contem um relatorio de vendas com uma base de dados de modelo da própria Microsoft.

* Featuires
  Dados de Navegação
  Mudança interativa entre os gráficos
  Indicadores
